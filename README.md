# Label press

Web application creating printable labels for archaeological finds from CSV input.

## Quick start

```bash
yarn install
yarn run dev
```

## Bundling

Use parcel for bundling, make sure to set the public path according to the deployment location.

```bash
parcel build --public-url /
```
